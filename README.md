# `xsocket-makefile`: Backup xsocket repository with Make support.

`xsocket-makefile` is a backup repository of [xsocket](https://github.com/koro666/xsocket) with a Makefile allowing non-Meson users to compile and install [xsocket](https://github.com/koro666/xsocket) easily.

See all the available targets with:

`cat ./Makefile`

This Makefile was generated with AI, but it is so functional that is ready to use.

Just type `make` and `sudo make install` and enjoy!
